CSS Pseudo-element

he css pseudo-element allow u to style the elements or parts of the elements without adding any ids or classes to them.

3 introduced a new double-colon (::) syntax for pseudo-elements to distinguished between them and pseudo-classes

selector::pseudo-element { property: value; }
