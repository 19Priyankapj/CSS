the css pseudo classes allow u to style the dynamic states of an element such as hover, active and focus state, as well as elements that are existing in the document tree but cant be targeted via the use of other selector 2without adding any ids or classezs to them, for example, targetting the first or last child element.

a pseudo-class starts with a colon (:), its syntax can be given with:

selector:pseudo-class { property: value; }
